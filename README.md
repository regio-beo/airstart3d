# Aistart 3d
High resolution air start analysis

## Setup data link
Create a symbolic link to the data directory 

```ln -s ~/Nextcloud/Paraglide/Competition-Pipeline data```

## Install
Install Python 3 and setup the correct dependencies. 

``` pip install -e .```

## Run
Run `py airstart3d/airstart_3d.py` to see how it works.

But you need the files from `pilot_progression` first!
